Política de Privacidad

Esta política de privacidad rige el uso de la aplicación móvil Horóscopo Semanal para dispositivos Android.

1. Información que recopilamos

Nuestra aplicación utiliza servicios de terceros que pueden recopilar información utilizada para identificarte:

  • Google Play Services: Para el funcionamiento básico de la app en Android.
  • Firebase Analytics: Recopilamos datos anónimos sobre el uso de la aplicación (por ejemplo, qué signos se consultan más) para mejorar la experiencia del usuario.
  • Firebase Cloud Messaging (FCM): Para enviarte notificaciones sobre el nuevo horóscopo semanal.

2. Servicios de terceros (YouTube)

Esta aplicación utiliza el reproductor de YouTube para mostrar contenido de video. Al utilizar este servicio, el usuario está sujeto a los Términos de Servicio de YouTube y a la Política de Privacidad de Google.

3. Cookies y Almacenamiento

La aplicación no utiliza "cookies" de manera directa, pero los servicios de terceros mencionados anteriormente pueden utilizarlas para recopilar información estadística.

4. Seguridad

Valoramos tu confianza al proporcionarnos acceso a datos analíticos anónimos. Nos esforzamos por utilizar medios comercialmente aceptables para protegerlos, pero recuerda que ningún método de transmisión por internet o almacenamiento electrónico es 100% seguro.

5. Cambios a esta Política

Podemos actualizar nuestra Política de Privacidad periódicamente. Se recomienda revisar esta página para detectar cualquier cambio.

6. Contacto
